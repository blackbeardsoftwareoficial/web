# Blackbeard Software

Sitio web oficial de Blackbeard Software - Activación y Soporte de Software.

## 🚀 Despliegue en GitHub Pages

1. Sube todos los archivos a tu repositorio de GitHub
2. Ve a **Settings** > **Pages**
3. En **Source**, selecciona la rama `main` y la carpeta `/ (root)`
4. Haz clic en **Save**
5. Tu sitio estará disponible en: `https://tu-usuario.github.io/tu-repo/`
