Política de Privacidad

Recolección de Datos

Recopilamos nombres, números de WhatsApp y, ocasionalmente, correos electrónicos. Importante: No almacenamos contraseñas de AnyDesk ni grabamos sesiones remotas sin aviso previo.

Finalidad

Los datos se usan exclusivamente para la prestación del soporte técnico solicitado y el envío de comprobantes de pago.

Seguridad

Utilizamos conexiones encriptadas de AnyDesk. Una vez terminada la sesión, el acceso se cierra y no podemos volver a ingresar sin un nuevo ID generado por el cliente.

Derechos

El cliente puede solicitar en cualquier momento la eliminación de sus datos de contacto de nuestra base de datos (Ley 1581 de 2012 - Habeas Data).

Términos de Servicio

Naturaleza del Servicio

Blackbeard Software presta servicios de soporte técnico, instalación y configuración de software comercial.

Modalidad

El servicio es 100% remoto mediante herramientas de control compartido. El cliente debe estar presente frente a su equipo durante la intervención.

Garantía

Ofrecemos una garantía de 30 días sobre la instalación realizada. Si el software presenta errores técnicos derivados de la instalación, lo corregimos sin costo adicional.

Responsabilidad

No nos hacemos responsables por la pérdida de datos previa a la intervención. Se recomienda al cliente realizar una copia de seguridad antes del soporte.

Reembolsos

Una vez iniciada la intervención técnica y lograda la ejecución del software, no se realizan reembolsos de dinero. Si por razones técnicas del equipo el software no se puede instalar, se devuelve el 100% del pago.
